# メモ

## let

- 変数(変更可能)

## const

- 定数(変更不可能）

## 条件分岐

- if文とswitch文の二つの方法

### if文

- 2つから片方を選択する場合に使用

### switch文

- 複数から選択する場合に使用

## 出力の中で比較演算子を使うとtrue or falseで帰ってくる

```js:エディタ
const age = 20
console.log(age > 10)
console.log(age > 30)
```

```console:ターミナル
true
false
```
## 反復処理

- while文とfor文の二つの方法

### while文

- 長いが文法がif文と同じ為、わかりやすい

### for文

- 変数の定義もfor文内ででき、シンプルに記述できる

## constructor(コンストラクタ)

- インスタンスを作る時、実行したい処理を中に記述

## this

- コンストラクタによって作成されたインスタンス

## extends

- 継承

## オーバーライド

- 親クラスを継承した子クラスで同名メソッドを上書きすること

## super

- constructorを継承する

## DOM

- JavaScriptのプログラムからHTMLを操作するための仕組み
