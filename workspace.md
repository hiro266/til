## 3. GitHub Flow
GitHub Flowで開発を進めていただきます。

### 1.作業ディレクトリへ移動して、下記コマンドでtopicブランチを作成

`git checkout -b xxx`

（xxxには各課題で指定されるブランチ名を入力してください）

### 2. 課題の要件に合わせて実装する

### 3. リモートリポジトリへpushします

例）

`git add .`

`git commit -m "[add]sorcery install"`

`git push origin xxx(ブランチ名)`

### 4. RUNTEQの課題画面にてプルリクエストを作成

### 5. 自動コードレビューを確認する
自動コードレビューが通るまで2-3を繰り返しましょう。

レビューにパスすると出てくるmergeボタンが出てきます

クリックして課題完了となります。

### 6. 自己採点
課題の提出後に確認ポイントと解答例が出てきます。必ず学習するようにしましょう。

### 7. 次の課題に進み1-6を繰り返す

### 8. 全ての課題が完了したらカリキュラム終了です
